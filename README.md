# atividade-extensionista-I-inclusao-digital
Repositório do projeto acadêmico da disciplina Atividade Extensionista I: Tecnologia Aplicada à Inclusão Digital – Levantamento. Desenvolvimento do protótipo da plataforma Veste Esperança, voltada para conectar doadores a comunidades carentes no Rio de Janeiro, alinhada aos ODS 1, 10 e 12.
